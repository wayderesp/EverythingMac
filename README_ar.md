## ✨ الميزات
- ⚡️ Dual-Engine Light-Speed Search
- 👁️ Ultimate UI Aesthetics with Glassmorphism
- 🛠️ Hardcore Geek Syntax (`ext:mp4`, `size:>1gb`)
- 🌍 9 Languages Supported

## 📦 التثبيت والاستخدام
1. Download `EverythingMac.dmg` from Releases.
2. Drag to `Applications`.
3. **IMPORTANT**: Grant `Full Disk Access` in System Settings.
4. Press `Cmd + Shift + Space` to start!
