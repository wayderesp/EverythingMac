🚀 **Bringing the ultimate file-searching experience of the Windows platform to every macOS user who values efficiency and aesthetics.**

EverythingMac is a highly mature global file search productivity tool. It breaks through the surface limitations of Spotlight and penetrates the macOS core, using native technologies to bring you a millisecond-level full-disk search experience. It is not just fast; it is breathtakingly beautiful.

## ✨ Features

- ⚡️ **Dual-Engine Light-Speed Search**: Fusing native `NSMetadataQuery` hardware acceleration with memory-level PCRE regular expression secondary filtering. Searching hundreds of thousands of files takes only a blink of an eye.
- 🧠 **Implicit Path Smart Engine**: No cumbersome commands needed. Simply type `/Downloads/ pdf` in the search box to instantly locate your desired downloads.
- ⌨️ **Global Shortcut Ejection**: Default `Cmd + Shift + Space` (customizable).
- 👁️ **Ultimate UI Aesthetics**: Perfectly supports SwiftUI declarative glassmorphism. Space-age interactive border glow that tracks your mouse.
- 🛠️ **Hardcore Geek Syntax**: `ext:mp4`, `size:>1gb`, `date:today`, `dir:/Users`, `regex:.*`.
- 🥷 **Resident & Incognito Modes**: Supports launching at login, and optionally hides the Dock icon completely.
- 🌍 **9 Languages Supported**: Natively supports English, Chinese, Japanese, Korean, French, German, Spanish, Russian, and Arabic.

## 📦 Installation & Usage

1. Download the latest `EverythingMac.dmg` from the Releases page.
2. Double-click to open, and drag `EverythingMac` into your `Applications` folder.
3. **IMPORTANT**: When opening for the first time, please go to `System Settings > Privacy & Security > Full Disk Access` and check the box for EverythingMac, otherwise it will only search a fraction of your files.
4. Press `Cmd + Shift + Space` to start your lightning-fast search journey!
5. In the app, you can adjust the zoom level, theme, and language in the settings.
