🚀 **将 Windows 平台的极速搜索神器体验，带给每一位注重效率与审美的 macOS 用户。**

EverythingMac 是一个高度成熟的全局文件检索效率工具。它打破了 Spotlight 的表层限制，穿透 macOS 底层，用最原生的技术为您带来毫秒级的全盘搜寻体验。

## ✨ 核心特性

- ⚡️ **双引擎光速检索**：原生 `NSMetadataQuery` 硬件加速 + 内存级 PCRE 正则二次过滤。
- 🧠 **隐式路径智能引擎**：随意输入 `/Downloads/ pdf` 瞬间定位。
- ⌨️ **全局快捷键弹射**：默认 `Cmd + Shift + Space`（可自定义）。
- 👁️ **极致 UI 审美**：支持 SwiftUI 声明式毛玻璃，以及极致的 Fluent Design 鼠标悬浮边框流光。
- 🛠️ **硬核极客语法**：`ext:mp4`, `size:>1gb`, `date:today`, `dir:/Users`, `regex:.*`。
- 🥷 **常驻与无痕模式**：开机自启，支持彻底隐藏 Dock 栏图标。
- 🌍 **九国语言支持**：原生内置。

## 📦 安装与使用方法

1. 从 Release 下载最新版 `EverythingMac.dmg`。
2. 将带有闪电图标的 `EverythingMac` 拖入 `Applications` 文件夹。
3. **重要提示**：首次打开时，请前往「系统设置 - 隐私与安全性 - 完全磁盘访问权限」为其打勾，否则只能搜到极少的文件。
4. 按下 `Cmd + Shift + Space` 即可唤醒搜索框！
5. 软件内支持 `Cmd + ,` 打开设置，进行快捷键、语言、主题调整。
